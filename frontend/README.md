## Overview
This is the frontend of my portfolio web application built with React. It showcases my projects, skills, and allows users to get in touch through a contact form.

The application is fully responsive and integrates with a backend API for dynamic content.

---

## Tech Stack
- React.js  
- JavaScript (ES6+)  
- HTML5 & CSS3  
- Tailwind CSS / Bootstrap  
- Axios  
- React Router DOM  

---

## Features
- Fully responsive design  
- Dynamic project rendering from backend  
- Smooth navigation with client-side routing  
- Contact form with API integration  
- Loading and error handling states  
- Clean and reusable component structure  

---

##  Backend Integration
This frontend communicates with a backend API for fetching and sending data.
